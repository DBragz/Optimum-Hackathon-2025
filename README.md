# Optimum Hackthon 2025

This is the main repository for team 1.

## Prerequisites

 - [Go 1.25.1+](https://go.dev/)

## Get Started

1. Build project.

```shell
go build receive.go
go build send.go
```

2. Run programs.

```shell
./receive    # Linux/macOS
receive.exe  # Windows
```

```shell
./send    # Linux/macOS
send.exe  # Windows
```
